CleanTrack AI is an intelligent platform designed to improve cleanliness management using AI-powered monitoring and reporting.

### Features

* Report cleanliness issues
* AI-assisted waste categorization
* Track complaint status
* Dashboard for monitoring reports
* User-friendly interface

### Tech Stack

* Next.js
* TypeScript
* MySQL
* AI APIs

### What I Learned

* Building AI-integrated applications
* Database management
* Modern frontend development
* API integration

### Future Improvements

* Real-time notifications
* AI image analysis
* Mobile application
* Advanced analytics dashboard

CleanTrack AI aims to make cleanliness management smarter, faster, and more efficient through technology.
