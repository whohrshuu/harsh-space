# 🚀 Harsh Space

A personal portfolio website built using **Next.js** to showcase my projects, skills, and development journey.

---

## 🌐 Live Features

- Home page with introduction
- Projects showcase page
- About page
- Resume page
- Responsive navigation bar
- Social media links (GitHub, LinkedIn, Instagram, X)

---

## ⚙️ Tech Stack

- Next.js (React Framework)
- TypeScript
- Tailwind CSS
- HTML / CSS / JavaScript

---

## 📂 Pages

- `/` → Home
- `/projects` → Projects
- `/about` → About me
- `/resume` → Resume / Experience

---

## 🚀 Deployment

This project is deployed using Vercel for fast and reliable hosting.

Live link: https://harsh-space.vercel.app



---

## 📸 Purpose

This project was built to improve my frontend development skills and understand real-world project structuring using React-based frameworks.
